# はじめまして！

## 記事の書き方

### Front Matter
記事の最初に以下のような情報を書きます：

```yaml
---
layout: post
title: "記事のタイトル"
date: YYYY-MM-DD
categories: [カテゴリ名]
tags: [タグ1, タグ2]
---
```

### マークダウン記法

#### 見出し
```markdown
# 大見出し
## 中見出し  
### 小見出し
```

#### リスト
- 項目1
- 項目2
  - サブ項目

#### コード
インライン： `System.out.println("Hello World");`

ブロック：
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
